# dr-appointment-website
A responsive single-page doctor website for booking online video consultations via WhatsApp.

## Features
- Online video consultation booking
- WhatsApp integration
- Clean and modern UI
- Fully responsive design
- Fast and lightweight (single HTML file)

## How to Use
1. Open the website
2. Click on “Book Video Consultation”
3. Redirect to WhatsApp with pre-filled message

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript

## 📞 Contact
For any queries, contact via WhatsApp.
